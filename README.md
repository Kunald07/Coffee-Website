# ☕ Delight – Coffee & Café Website

A modern and visually engaging coffee shop landing page built using **HTML, CSS, and JavaScript** with smooth **GSAP animations**. The project showcases a premium café experience through elegant layouts, interactive sections, and responsive design elements.

---



## ✨ Features

- ☕ Elegant coffee shop landing page
- 🎨 Modern and aesthetic UI design
- 🚀 Smooth GSAP animations
- 📜 Scroll-triggered effects
- 🍰 Beautiful menu and specials section
- 📷 Image gallery slider
- 📱 Responsive layout
- 📧 Newsletter subscription section
- 🛒 Shopping cart and navigation icons
- 🍯 Featured products section

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- GSAP (GreenSock Animation Platform)
- ScrollTrigger
- Remix Icons
